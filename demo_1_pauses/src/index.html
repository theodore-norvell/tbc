Demo 1. This is based on my blog entry. This one demonstrates timeouts and pauses.

The TBC code is as follows.

       loop(
            nag( null ) >
            await(
                click( b1a ) >> out1A
            ,
                click( b1b ) >> out1B
            ,
                timeout( 2000 ) >> tooLate
            ) >
            pause(1000) >
            await( click( b2 ) >> out2 )
        )

where the nag function is defined by

       static function nag(triv : Triv) : Process<Triv>{ return
           await(
               click(b0) && exec(thankTheUser)
           ,
               timeout( 1000 ) && exec(nagTheUser) >= nag
           ) ; }
